# Demos

Working examples of the Neo4j Orchestration Framework.

## Quick Start (1 minute)
```bash
python demos/quick_start.py
```

Shows the complete pipeline in action.

## Complete Week 3 Pipeline (5 minutes)
```bash
python demos/week3_complete_pipeline.py
```

Demonstrates:
- Natural language classification
- Cypher query generation
- Query execution with results
- Error handling

## Requirements
```bash
pip install -e .
```

All demos use mocked Neo4j connections, so no database required.
